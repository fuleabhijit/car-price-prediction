This project uses historical car data and a trained Random Forest Regression model to estimate the resale value of a car.

Users provide:

- Car Name
- Year of Purchase
- Present Price
- Kilometers Driven
- Fuel Type
- Seller Type
- Transmission
- Number of Owners

The system processes this data and returns a predicted selling price in lakhs.
